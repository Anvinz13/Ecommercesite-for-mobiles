# Ecommerce Site for Mobiles

**Ecommercesite-for-mobiles** is an ASP.NET-based ecommerce web application designed for selling mobile phones and accessories.  
The project demonstrates core ecommerce functionalities such as product listing, browsing, and basic order flow.

## Project Overview

This project is developed as a learning and academic project using **ASP.NET**.  
It focuses on building a simple ecommerce platform for mobile phones, covering both frontend and backend concepts.

---

## Features

- Mobile phone product listing  
- Product details page  
- Basic ecommerce workflow  
- Structured backend logic using ASP.NET  
- Database connectivity for product storage (if implemented)
